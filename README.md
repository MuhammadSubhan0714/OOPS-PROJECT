# OOPS-PROJECT
A console-based strategy card game inspired by Clash Royale, built in C++. The project simulates core gameplay mechanics including card battling, player progression, an in-game shop, and clan game events. It serves as a comprehensive demonstration of major Object-Oriented Programming (OOP) concepts.
1. Battle System: Players use a deck of cards to deploy troops, spells, and
buildings against an opponent.
2. Player Progression: Players earn trophies, level up, and collect coins through
battles and achievements.
3. In-Game Shop: A shop system allows players to purchase new cards using in-
game coins.
4. Dynamic Card System: Supports different card types (Troop, Spell, Building),
each with unique behaviors.
5. File Handling: Player progress (trophies, coins, collection) is saved and loaded
using file streams (e.g., player.txt).
6. Robust Error Handling: Manages edge cases like insufficient funds for
purchases using exception handling.
